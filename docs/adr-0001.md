# ADR-0001: Mini Landing Mimari Kararlari

- Tarih: 2026-02-28
- Durum: Kabul Edildi

## Baglam
Gorev, tek sayfa urun tanitim landingi ve kucuk bir bilesen kutuphanesi talep ediyor.
Harici UI kutuphanesi kullanilmamali, SCSS zorunlu, 3 breakpoint ve temel a11y
kriterleri saglanmali.

## Karar
- UI katmani React + TypeScript ile kuruldu.
- Stiller SCSS ve BEM sinif adlandirmasi ile yazildi.
- Tema yonetimi `data-theme` + CSS variables ile uygulandi.
- Bilesenler `src/components`, sayfa bolumleri `src/sections` altinda ayrildi.
- Form dogrulamasi ek kutuphane olmadan yalnizca React state ile yapildi.

## Sonuclar
- Bilesen tekrar kullanimi artar.
- Temalar arasi gecis merkezi renk degiskenleri ile kolaylasir.
- Kod tabani buyudugunde section ve component ayrimi bakimi kolaylastirir.
